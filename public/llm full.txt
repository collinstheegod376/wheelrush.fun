Wheel Rush SEO Context

Site type

Current SEO files

Primary SEO goals

SEO risk areas

- `index.html` title and description must stay clean and uncorrupted
- Open Graph and Twitter tags should match the visible branding
- The OG image should be a proper social-share asset, not just the app icon if a better preview image exists

Keyword direction

- spin the wheel game
- prize wheel game
- spin to win
- free online jackpot game
- virtual rewards game
- leaderboard game

Recommended page intent
- Homepage should explain the game in one sentence
- Add a short explainer section if organic search is a priority
- Keep copy aligned with "free virtual rewards" rather than cash gambling claims

Technical checks
- `public/robots.txt` allows crawling and points to sitemap
- `public/sitemap.xml` currently lists the root URL only
- PWA manifest is enabled and should not conflict with crawlable content
- Canonical domain must match the deployed production URL

Cross-check items
- Confirm the title in `index.html` renders without encoding corruption
- Confirm description length is reasonable for search snippets
- Confirm social tags are present and point to a reachable image
- Confirm `sitemap.xml` uses the current `lastmod`
- Confirm the deployed hostname matches `canonical` and `og:url`

Notes for future SEO work
- If you add marketing pages, update sitemap and internal links
- If you add FAQs, include them in HTML so search engines can read them
- If you add a hero section, make sure it includes the primary keyword naturally
- Avoid keyword stuffing; keep the language focused on free play and virtual rewards
